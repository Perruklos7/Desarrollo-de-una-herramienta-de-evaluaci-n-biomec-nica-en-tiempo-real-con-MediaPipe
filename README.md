# Desarrollo-de-una-herramienta-de-evaluaci-n-biomec-nica-en-tiempo-real-con-MediaPipe
En esta práctica se propone el desarrollo de un sistema de análisis biomecánico en tiempo real utilizando técnicas de visión por computador. En particular, se empleará la librería MediaPipe Pose para la detección de puntos anatómicos (landmarks) a partir de la imagen capturada mediante una webcam
